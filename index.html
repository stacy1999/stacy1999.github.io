My Final Project




During the last week at the Generation BootCamp, I worked on building a Task Planner web application using JavaScript. The program would take in user inputs through a form, validate the data against prerequisites and populate the page with a card with the information passed through. Finally, it also allows you to delete the task or update it with new inputs. I tracked its progress and changes with the use of Git and if you'd like to find out more about how it was built you can find it on my GitHub profile.
